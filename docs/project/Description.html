Short Description

In my application are the users able to get an overview of the information of their cat.

They can add, delete and edit cats, informationlist and their informations.



Feature Description
Security

Security system using the default security plugin for Grails.

The login for the admin is:

  • Username: admin
  • Password: 1234

The login for the user is:

  • Username: user
  • Password: 1234

The admin is allowed to add new people in the household while the admin is not.

Both have the same permissions for the rest of the application.
